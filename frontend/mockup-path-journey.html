Stage 5 · Post-session reflection
Euler reflects on the session → adapts the path
After every node, a lightweight reflection agent (Haiku tier) runs. It reads the session's notes + signals + assessment → writes path_notes with strengths/gaps → may propose pivots (insert remediation, skip ahead, swap a node). User confirms before path mutates — keeps trust.
