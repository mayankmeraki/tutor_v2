Diff — code bug fix
before
while lo < hi: mid = (lo + hi) // 2 if nums[mid] == target: return mid elif nums[mid] < target: lo = mid + 1 else: hi = mid - 1
after
while lo <= hi: mid = lo + (hi - lo) // 2 if nums[mid] == target: return mid elif nums[mid] < target: lo = mid + 1 else: hi = mid - 1
Two fixes: < → <= (check last element) and (lo+hi)//2 → lo+(hi-lo)//2 (overflow safe).
Diff — misconception correction
misconception
Heavier objects fall faster because gravity pulls them harder.
reality
More mass = more gravitational force, BUT also more inertia. The two cancel exactly. All objects fall at 9.8 m/s² regardless of mass.
Diff — SQL query improvement
slow
SELECT * FROM orders WHERE YEAR(created_at) = 2025
fast
SELECT * FROM orders WHERE created_at >= '2025-01-01' AND created_at < '2026-01-01'
Left wraps every row in YEAR() — can't use the index. Right is a simple range scan on the indexed column.
